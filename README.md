clasdigi : Clas12 digitizer
===========================

Installing
----------

###Requirements

- Modern C++ compiler with c++11/14 support
- CLHEP
- ROOT6

###Building

```
git clone git@github.com:whit2333/clasdigi.git
mkdir clasdigi_build && cd clasdigi_build
cmake ../clasdigi/. -DCMAKE_INSTALL_PREFIX=$HOME
make install
```
