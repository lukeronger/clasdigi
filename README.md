clasdigi : Clas12 digitizer
===========================

Installing
----------

```
git clone it@github.com:whit2333/clasdigi.git
mkdir clasdigi_build && cd clasdigi_build
cmake ../clasdigi/. -DCMAKE_INSTALL_PREFIX=$HOME -DPROJECT_USE_ROOT6=1
make install
```
